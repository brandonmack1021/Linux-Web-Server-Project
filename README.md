# Linux-Web-Server-Project
Hands-on project deploying an Ubuntu Linux web server on AWS EC2 with Nginx, firewall configuration(UFW), and basic Linux administration.

## What I learned 

- How to **launch and connect to an Ubuntu EC2 instance** on AWS.
- Using **SSH** for secure remote access.
- Installing and managing **Nginx web server**.
- Configuring **UFW firewall rules** safely for SSH and HTTP.
- Understanding **file structure and index precedence** in web servers.
- Deploying a **custom static webpage** on a Linux server.
- Basic Linux administration commands for package updates, service management, and system monitoring.
- Importance of **documentation and repeatable workflows**.
